Title: Dark Mode Toggle with Smooth Animation

Description:
This project is a beautifully animated Dark Mode toggle switch built with React and TypeScript. It features smooth transitions, interactive hover effects, and a unique sun-to-moon transformation. The sun’s beams rotate on hover, and clicking the toggle smoothly transitions between light and dark mode. The design is crafted using CSS Modules for better styling encapsulation.

Features:

🌞 Sun-to-moon transition with a masking effect

🎨 Animated sunbeam rotation on hover

⚡ Smooth dark mode activation

🛠️ Built with React, TypeScript, and CSS Modules
